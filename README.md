# Cipher
Cipher is a game made for a modded 3ds. It follows the story of Mello, who fell off a cruise and turned into an eel.
## Features
* Cipher has multiple perspectives for each chapter. At the start of each chapter, the player will choose what available character to play as.
* When you complete the game, you will be able to find CDs explaining the story.
* Features 3D low-poly graphics.
* Both screens are used.

## I want to make a mod. Can I do so?
Yes, but under the following circumstances:

* You do not monetize the mod
* You keep everything age appropriate (maximum PG-13)
* The mod is not intended to cause hate on another individual
## Can I use your code/assets to build my game?
Not without permission. Contact me first.
## Will there be voice acting?
No. This is because I cannot afford to hire people, and the file size needs to be managable.
## Can I create fan art?
Yes, with the same rules as for modding.
## I want to make content on Cipher.
You may, please read acceptable and unacceptable use of making content.
### Acceptable:
* Fan animations
* Let's play videos
* Reviews

### Not Acceptable:
* Animated series
